# SyncNote

A web app for writing, organizing, and sharing notes with Markdown support.

> **Status:** Frontend complete — backend (auth, persistence, sharing) in progress.

## Features

- Markdown editor with live preview
- Note organization with sidebar navigation
- Share notes via link (UI ready)
- Settings & profile modals
- Clean, responsive UI

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS

## Roadmap

- [ ] User authentication
- [ ] Database integration (notes persistence)
- [ ] Real share-link backend
- [ ] Public note viewer

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
